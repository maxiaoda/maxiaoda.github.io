如何用hugo搭建个人博客

本人搭建环境：

  1. windows10 操作系统；
  2. 安装Git，version-2.25.0；
  3. 安装hugo，version-v0.65.2；

搭建步骤如下：

  1. 安装Hugo和添加环境变量：

    在Hugo的官方Github中，找到 Releases 下载对应的操作系统版本。将安装目录添加至windows系统环境变量中。

  2. 使用命令新建一个网站（使用命令时会在当前目录生效）。

 hugo new site SiteName
  1. 进入SiteName目录cd SiteName输入以下命令，添加一个主题。
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
  1. 将主题添加至config.toml中。
echo 'theme = "ananke"' >> config.toml
  1. 添加一篇名为my-first-post的博客。
hugo new posts/my-first-post.md
  1. 在my-first-post文件头部中的draft改为false即可发布博客。
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
  1. 启动Hugo server
hugo server -D

成功后访问http://localhost:1313/

  1. 配置config.toml，baseURL设为新域名，languageCode的中文为zh-Hans，title为自己的博客名称。
baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

9.建立一个静态页面。

hugo -D

可生成public目录。
